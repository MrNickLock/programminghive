```
TINYINT

FLOAT

STRING

BINARY
```

## 3.2 集合数据类型

```
STRUCT
MAP
ARRAY
```

## 3.3 文本数据编码

```
\n  文本文件,分隔行
^A (“control” A) 列分隔字符,用八进制\001表示
^B 用于分隔```Array```或```Struct```中的元素或者```Map```中的键值对,用八进制\002表示
^C 用于```Map```中键和值之间的分隔,用八进制\003表示
```
